# hugo-mcp-go v0.1.0

## Summary

`hugo-mcp-go v0.1.0` is the release candidate that introduces the native Go HTTP backend for Hugo MCP while preserving stdio for rollback and local use.

Included in this release:

- native Go HTTP backend
- stdio preserved
- chunking and pagination for large page and asset content
- native SRI checks
- MCP tool hints for better client classification
- production validation on the controlled native path

## Rollback

Rollback remains preserved:

- native HTTP backend can be removed from the active path
- shim backend remains installed
- Python backend remains installed

## Upgrade notes

For native backend deployments:

- set `HUGO_MCP_TRANSPORT=http`
- configure `HUGO_MCP_HTTP_TOKEN_FILE`
- point `HUGO_MCP_URL` to the native backend endpoint
- keep `mcp-runtime-go` in place for public OAuth and gateway handling
